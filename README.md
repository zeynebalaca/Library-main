# Library – Kütüphane Yönetim Sistemi

Bu proje, ASP.NET Web Forms ve MSSQL kullanılarak geliştirilmiş bir kütüphane yönetim sistemi prototipidir.
Sistem, kullanıcı ve admin rollerine göre farklı yetkiler sunacak şekilde tasarlanmıştır.

## Kullanıcı İşlevleri
- Giriş / Kayıt: Kullanıcılar e-posta ve şifre ile sisteme giriş yapabilir veya yeni hesap oluşturabilir.
- Kitaplar: Kütüphanedeki tüm kitaplar listelenir, kitap detayları ve kullanıcı yorumları görüntülenir.
- Ödünç Alma: Uygun durumdaki kitaplar ödünç alınabilir.
- Benim Ödünçlerim: Kullanıcının aldığı ödünçler tarih ve durum bilgileriyle görüntülenir.
- Benim Yorumlarım: Kullanıcı kendi yorumlarını görüntüleyebilir, düzenleyebilir veya silebilir.

## Admin İşlevleri
- Kitap Yönetimi: Kitap ekleme, güncelleme ve silme işlemleri.
- Kategori Yönetimi: Kategori ekleme, yeniden adlandırma ve silme.
- Kopya Yönetimi: Kitap kopyalarının yönetimi.
- Ödünç Yönetimi: Ödünç alma ve iade işlemlerinin takibi.
- Admin Dashboard: Sistem istatistiklerinin özet görünümü.
